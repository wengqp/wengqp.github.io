安装

安装 nodeJS

安装 Hexo

在 cmd 命令行中输入：

hexo server 也可以缩写成 hexo s，加入 --debug 参数可以看到具体加载信息。
按 Ctrl+C 退出服务。

安装NexT

下载 Hexo 主题 NexT，解压并改名为 next，放到 blog 文件夹中的 themes 下。
blog 项目中有两个地方有 _config.yml（配置文件），blog/ 下的是站点配置文件，blog/themes/next/ 下的是主题配置文件。

更换主题

用 nodepad++ 打开站点配置文件，找到 theme 字段，改为：

theme: next

重新启动服务就可以看到效果，否则执行 hexo clean 命令清理缓存再重启试试。

更改站点配置文件

#简体中文

动手写篇博文

修改站点配置文件：

创建新 md 文件时，文件名自动加上日期前缀，且空格被短横替代，例如: “2016-10-27-hello-world.md”

new_post_name: :year-:month-:day-:title.md
变量 描述
:title 标题（小写，空格将会被替换为短杠）
:year 建立的年份，比如， 2015
:month 建立的月份（有前导零），比如， 04
:day 建立的日期（有前导零），比如， 07
